# Generating Anki Deck

```shell
$ python make_anki_deck.py \
  --csv=lesson_5_vocabulary_and_holidays.csv \
  --audio-dir=lesson_5_audio \
  --deck-title='Fuji Lesson 5 Vocabulary' \
  --deck-filename='fuji_lesson_5'
```
